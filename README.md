# Cheaper Choice

### This app is part of the 'SPA per WEEK' project
### Give it your travel start and end points, and CheaperChoice will retrieve the latest price estimate from Über and plot your route on the map.

## Technologies Used
- Node / Express

## Getting Started
- Fork / Clone
- Run `npm install`
- Run `bower install`
- Set up a .env file for your Über keys.
